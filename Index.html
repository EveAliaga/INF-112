Un ingeniero programador

Los primeros lenguajes de programacion

El viaje de cualquier ingeniero que desee programar empieza con el aprendizaje de uno o varios lenguajes de programacion. Exite una gran variedad de lenguajes y de herramientas que facilitan el aprendizaje. Sin embargo, la practica no puede desligarse de la teoria. Es decir, es necesario conocer tambien los diagramas de flujo y como se interpretan en cada lenguaje.

Lenguajes que convivien en el entorno

Actualmente existen diversos lenguajes de programacion que conviven y en muchos casos se combinan para obtener resultados, algunos de ellos son:

  • Python
  • Java
  • HTML
  • PHP
